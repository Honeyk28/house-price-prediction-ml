# House Price Prediction using Linear Regression

## Project Overview

This project uses Machine Learning to predict house prices using the California Housing dataset. A Linear Regression model was trained and evaluated using various performance metrics.

---

## Objective

The objective of this project is to understand the complete Machine Learning workflow including:

- Data loading
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training
- Prediction
- Model evaluation

---

## Dataset

The California Housing dataset from scikit-learn was used in this project.

Features include:
- Median Income
- House Age
- Average Rooms
- Population
- Latitude and Longitude

Target:
- Median House Price

---

## Evaluation Metrics

The model was evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
